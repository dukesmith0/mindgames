# mindgames

Lightweight shared scoreboard for three friends to play brain games — zetamac-inspired minimal UI.

**Live site:** https://www.craigdukesmith.com/mindgames/

## Games
- **Speed Math** — 60s zetamac-style drill (+, −, ×, ÷)
- **Digit Span** — memorize and recall digit sequences
- **Reaction Time** — 5-trial wait-for-green test
- **Minesweeper** — 10×10 grid, 15 mines, timed
- **Word Recall** — 20s to memorize 10 words, then recall

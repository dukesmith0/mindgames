# mindgames

Lightweight shared scoreboard for three friends to play speed math, sudoku, and reaction time — zetamac-inspired minimal UI.

**Live site:** https://dukesmith0.github.io/mindgames/

## Games
- **Speed Math** — 60s zetamac-style drill (+, −, ×, ÷)
- **Sudoku** — 9×9 timed, unique-solution generator
- **Reaction Time** — 5-trial wait-for-green test
